  • Hi Chief! Welcome to Clash Royale Card Maker!

    Clash Royale is a freemium mobile strategy video game developed and published by Supercell. The game combines elements from collectible card games, tower defense, and multiplayer online battle arena. The game was released globally on March 2, 2016.

    Thanks to this simple tool, you can create your own Clash Royale card! You can use it to give a better idea when you create a new card for Reddit or even just for fun!

    This tool was created by Giorgio Garasto, a Software Engineer who loves making websites and also loves Clash Royale. He initially built the tool just after Clash Royale was, released, when he was just 19 years old. If you like its work, please consider donating using the button you can find in the chat tab.

    Thank you!

    • 3.0.0-beta.0

      2025-10-15

      New implementation from scratch based on the experimental 2.0.0b version.
      This new version has a lot of new features:

      • On-card form editor
      • New template based on modern Clash Royale cards
      • Multi-page cards support
      • Multiple templates support (the OG Clash Royale template is still supported!)
      • Possibility to share directly from the UI (if your browser supports it)
      • Possibility to save and view the cards locally in your browser
      • Possibility to install and use the Card Maker without an internet connection!

      This is still a beta, but a lot of new features are planned for the future, only possible thanks to the new architecture:

      • Possibility to load back and edit cards saved in the browser
      • Possibility to create also Champions, Tower Troops, and Rulers
      • Custom templates
      • And much more!

      Also, some features available in version 1.x.x are not yet implemented, but they will be added soon:

      • Web app language selection
      • Card language selection
      • JSON import/export

      Oh and by the way, the project is now open source! You can find the code on my GitHub.
